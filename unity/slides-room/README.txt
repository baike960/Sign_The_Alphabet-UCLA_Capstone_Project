The three files under the current directory are used to create the tutorial room that creates slides to demonstrate the picture of each letter sequentially. Acquired the source files from the unity asset store.

BaseMenu.cs-this file creates the main menu of the tutorial room.

CanvasMenu.cs-this file creates the slides for gesture pictures.

SwipeEffect.cs-this file smoothens the transition between each slide by implementing the swipe effect when the images are changed.
